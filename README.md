# rss-react-task

1. Clone repository
```
$ git clone https://github.com/lobzhaY/rss-react-task.git
```
2. Change branch
```
$ git checkout week6-ssr
```
3. Install dependencies
```
$ npm install
```
4. Start lint
```
$ npm run lint
```
5. Start tests coverage
```
$ npm run coverage
```
6. Start E2E tests
```
$ npm run test:cypress
```
7. Start dev server
```
$ npm run dev
```
